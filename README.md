## 说明：
#### 使用支付宝或微信支付功能时，需要把相关的APPID等信息替换为自己的。然后微信支付包名还需要改成自己申请的，推荐一种快速修改微信包名的方式：在gradle中把applicationId改成自己的，然后创建相应的包名并添加wxapi文件夹用自己的keystore打包就可以了。
